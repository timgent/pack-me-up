# Pack Me Up

Pack Me Up is a front-end only web application built with ReactJS that helps people generate customised packing lists.

You can find the site at <https://packmeup.tim-gent.com/>

# Getting started

```bash
npm install
npm run dev
```

If you want to test locally with solid you can run a local Solid server with:

```bash
npx --yes @solid/community-server -p 4000
```

Running tests:

```
npm test
```
